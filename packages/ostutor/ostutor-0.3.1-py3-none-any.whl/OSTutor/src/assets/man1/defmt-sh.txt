NAME
       defmt-c,  defmt-sh  - a sample ispell deformatter for C, C++, and shell
       sources

SYNOPSIS
       ispell -F defmt-c [ispell_options]
       ispell -F defmt-sh [ispell_options]

DESCRIPTION
       The defmt-c and defmt-sh programs are sample deformatter provided  pri-
       marily to show how to write filters for use with ispell(1)'s -F switch.
       They allow ispell(1) to be used to check C and C++ programs as well  as
       shell scripts (and any other languages that use # for comments and both
       quote styles for strings).
