NAME
       py_compilefiles - byte-compile python .py files

SYNOPSIS
       py_compilefiles [options] [file ...] [-]

DESCRIPTION
       py_compilefiles  compiles  Python  .py  source  files into .pyc or .pyo
       bytecode format.

OPTIONS
       -f     force rebuild even if timestamps are up-to-date.

       -i     ignore errors on byte compilation.

       -q     quiet operation.

       -x regexp
              skip files matching the regular expression regexp the regexp  is
              search for in the full path of the file.

       If  a  filename is given as a minus sign (-), a list of files separated
       by newline is read from standard input.
